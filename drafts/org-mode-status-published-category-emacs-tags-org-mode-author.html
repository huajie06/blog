Some settings

  1. infile options

    • #+todo: A B C | D E
    • #+startup: showall
    • #+author and etc.

  2. Refresh the setting at top with C-c C-c

  3. Keys

    • insert a header C-Enter
    • insert a header with todo S-C-Enter
    • toggle a header with todo S-left/right
    • complete M-tab
    • insert date/datetime C-c ., and C-u C-c .

  4. Some convention

    • Paragraph alignment like this.

  5. Markups

    • Like following

      {.example} *bold*, /italic/, _underlined_, =code= and ~verbatim~, and, if you must, +strike-through+.

  6. Add a tag

    • Use C-c C-c when are a header
    • Say if the TAGS are the following, can use hotkeys to pick #+TAGS: COMPUTER(c) PHONE(p) READING(r)

  7. Planning

    • insert a schedule at current item C-c C-s
    • insert a deadlin at current item C-c C-d
    • org-agenda to list all items
      • in order to display need to have todo and time items
