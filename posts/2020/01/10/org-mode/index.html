Some settings

Infile options

  • #+todo: A B C | D E
  • #+startup: showall
  • #+author and etc.

Refresh the setting at top with C-c C-c

Keys

  • insert a header C-Enter
  • insert a header with todo S-C-Enter
  • toggle a header with todo S-left/right
  • complete M-tab
  • insert date/datetime C-c ., and C-u C-c .

Some convention

  • Paragraph alignment like this.

Markups

  • Like following
*bold*, /italic/, _underlined_, =code= and ~verbatim~, and, if you must, +strike-through+.

Add a tag

  • Use C-c C-c when are a header
  • Say if the TAGS are the following, can use hotkeys to pick

#+TAGS: COMPUTER(c) PHONE(p) READING(r)

Planning

  • insert a schedule at current item C-c C-s
  • insert a deadline at current item C-c C-d
  • org-agenda to list all items
    • in order to display need to have todo and time items
