Python News Scraping Notes

  • Thu, 2019-12-26

The intention of this site was really for myself to have a place to scan a collection of news quickly.

The goal was to use Python to

  • Request a news website xml/html feed
  • Parse it using some library, keep the wanted info
  • Store it to database
  • Dashboarding

Collection of news source

  • CNBC
  • Bloomberge, subscription needed
  • Yahoo
  • WSJ, subscription needed
  • ~~Reuters~~
  • MarketWatch
  • Google Search with China as a key words

The script will runs every 10 min and there's a random number generator to determine if the code gets executed within each source block.

The notes area use Javascript Showdown to convert markdown into HTML rendering.
